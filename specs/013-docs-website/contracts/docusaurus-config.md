```typescript
  docs: [
    {

    },
  ],

```

## Section Category Metadata (`_category_.json`)

Each docs subdirectory MUST contain a `_category_.json` with at minimum `label` and `position`:

| Directory | `label` | `position` |
|-----------|---------|------------|
| `getting-started/` | `Getting Started` | `1` |
| `configuration/` | `Configuration` | `2` |
| `commands/` | `Commands` | `3` |
| `guides/` | `Guides` | `4` |
| `architecture/` | `Architecture` | `5` |

## Documentation Page Frontmatter Contract

Every documentation page MUST include frontmatter with at minimum:

```yaml
---
title: <page title>
sidebar_position: <integer>
---
```

Optional fields:
```yaml
---
description: <SEO description>
sidebar_label: <override sidebar display text>
---
```
